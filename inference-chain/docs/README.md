## Ключевые концепции

### Модули

Inference-chain состоит из **7 кастомных модулей**:

| Модуль | Назначение | Документация |
|--------|-----------|--------------|
| **BLS** | Пороговые подписи и DKG | [Архитектура](./architecture-overview.md#1-bls-xbls) |
| **Bookkeeper** | Двойная бухгалтерия | [Архитектура](./architecture-overview.md#2-bookkeeper-xbookkeeper) |
| **Collateral** | Залоги и слэшинг | [Архитектура](./architecture-overview.md#3-collateral-xcollateral) |
| **GenesisTransfer** | Перенос genesis-аккаунтов | [Архитектура](./architecture-overview.md#4-genesistransfer-xgenesistransfer) |
| **Inference** | Основная логика (670 файлов) | [Архитектура](./architecture-overview.md#5-inference-xinference---основной-модуль) |
| **Restrictions** | Ограничения переводов | [Архитектура](./architecture-overview.md#6-restrictions-xrestrictions) |
| **StreamVesting** | Vesting с разблокировкой по эпохам | [Архитектура](./architecture-overview.md#7-streamvesting-xstreamvesting) |

### Жизненный цикл эпохи

```
Создание → Генерация PoC → Обмен PoC → Валидация PoC →
Расчет весов → Переключение валидаторов → Активная эпоха →
Обслуживание инференсов → Следующая эпоха
```

**Детали:** [Временная линия эпохи](./proof-of-work-2.0.md#временная-линия-жизненного-цикла-эпохи)

### Workflow инференса

```
Запрос → Выбор исполнителя → Выполнение →
Приватный выбор валидаторов → Валидация →
Консенсус → Вознаграждения
```

**Детали:** [Механизм валидации](./cli-and-validation.md#механизм-валидации-инференса)

## Технологический стек

### Blockchain
- **Cosmos SDK** v0.50+ - Фреймворк для блокчейна
- **CometBFT** v0.38.17 - Byzantine-fault tolerant консенсус
- **Protocol Buffers** - Сериализация данных
- **IBC** - Межцепочное взаимодействие

### Криптография
- **BLS Signatures** - Пороговые подписи
- **ECDSA (secp256k1)** - Подписание транзакций
- **Ed25519** - Консенсус-ключи валидаторов
- **SHA256** - Хэширование для PoC

### Хранилище
- **Collections Framework** - Типобезопасное управление состоянием
- **LevelDB / RocksDB** - Физическое хранилище
- **In-Memory Cache** - Оптимизация производительности
